QR-Found
Веб-сервіс для створення персоналізованих QR-кодів, призначених для пошуку та повернення загублених речей власникам.

Функціональні можливості
1.Генерація динамічних QR-кодів без використання сторонніх API.
2.Серверний рендеринг (SSR) сторінок на базі шаблонізатора Jinja2.
3.Система реєстрації контактних даних власника (телефон, email, Telegram).
4.Адаптивний інтерфейс, оптимізований для мобільних пристроїв.

Технологічний стек
1.Backend: Python 3.12, Flask.
2.Frontend: HTML5, CSS3, Jinja2
3.Libraries: qrcode (PIL), gunicorn.
4.Deployment: Render.com.

Архітектура
Проєкт базується на монолітній структурі. Користувач заповнює форму, дані зберігаються у внутрішньому сховищі сервера, після чого генерується унікальний ідентифікатор та відповідний QR-код. При скануванні коду сервер динамічно формує сторінку з контактами власника.

Встановлення та запуск
Для локального розгортання виконайте наступні команди:

1. Клонування репозиторію:
Bash
git clone https://github.com/yellowmedok/qr_found.git
cd qr_found

2. Встановлення залежностей:

Bash
pip install -r requirements.txt
Запуск застосунку:

Bash
python app.py
